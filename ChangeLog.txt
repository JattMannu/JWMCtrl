04/04/2015 - 1.0.0-beta4
  1. Add option '-Y <WIN>' to iconify (minimize) the window.
  2. Add option '-S' to list windows in stacking order (bottom to above).
  3. Add option '-j' to list current desktop.
  4. Add option '-r <WIN> -y <MVARG>' to resize and move the window around
     the desktop, then reactivate. 
  5. Add option '-E <WIN>' to show the title of the window.
  6. Fixed bugs for 64bit system.
  7. Fixed bugs when use '-i' option to interpret <WIN> as a numerical
     window ID.
